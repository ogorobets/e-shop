# e-shop
Simple e-shop
<h2>Installation<h2>
<p>
	Run 'npm install'<br/> 
</p>
<h2>Starting app<h2>
<p>
	1. Install python (if you don't have one now) <br/> 
	2. Open a terminal and go to project folder <br/> 
	3. Run 'python -m http.server 8080' <br/> 
	4. Open 127.0.0.1:8080 url in browser <br/> 
</p>
